Ключевые темы экзамена
Тема
Что нужно знать чётко
Subnetting
Считать на скорость: сколько хостов, адрес сети, broadcast. VLSM без калькулятора.
STP root bridge
Выбирается по наименьшему Bridge ID (priority + MAC). Portfast только на access-портах.
OSPF DR/BDR
Выбирается по highest router-id / highest loopback / highest interface IP. Broadcast vs P2P.
ACL порядок правил
Первое совпавшее правило применяется. Implicit deny в конце. Стандартный ACL — ближе к dst.
NAT PAT
ip nat inside/outside на интерфейсах. overload = PAT. Проверить show ip nat translations.
HSRP vs VRRP
HSRP: Cisco proprietary, active/standby. VRRP: open standard, master/backup. GLBP: load balancing.
802.1Q vs ISL
802.1Q — стандарт, 4-байт тег, native VLAN не тегируется. ISL — устаревший Cisco, инкапсуляция.
WPA2 vs WPA3
WPA3 — SAE вместо PSK, обязателен для Wi-Fi 6. WPA2 — AES/CCMP. WEP = уязвим, не использовать.
