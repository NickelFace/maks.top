## Что нужно знать

Согласно официальным целям LPIC-1:

- Базовая настройка CUPS (для локальных и сетевых принтеров).
- Управление очередями печати пользователей.
- Диагностика общих проблем печати.
- Добавление и удаление заданий в настроенных очередях принтеров.

Ключевые файлы и команды: файлы конфигурации CUPS, `/etc/cups/`, `/var/log/cups/`, `lp`, `lpr`, `lpq`, `lprm`, `lpstat`, `lpadmin`, `cancel`, `cupsenable`, `cupsdisable`, `cupsreject`, `lpmove`.

## CUPS — Common Unix Printing System

CUPS (Common Unix Printing System) — стандартная система печати в современном Linux. Демон CUPS — `cupsd`.

### Файлы конфигурации CUPS

| Файл | Описание |
|---|---|
| `/etc/cups/cupsd.conf` | Основная конфигурация демона CUPS |
| `/etc/cups/printers.conf` | Список настроенных принтеров (не редактировать при работающем CUPS) |
| `/etc/cups/ppd/` | PPD-файлы (PostScript Printer Description) для каждого принтера |
| `/etc/printcap` | Файл совместимости с LPD, генерируется CUPS |

Чтобы включить веб-интерфейс, добавьте в `cupsd.conf`:

```
WebInterface Yes
```

### Веб-интерфейс

CUPS предоставляет веб-интерфейс по адресу `http://localhost:631`.

### Файлы журналов

| Файл | Описание |
|---|---|
| `/var/log/cups/access_log` | HTTP-запросы к веб-интерфейсу CUPS |
| `/var/log/cups/page_log` | Каждая напечатанная страница |
| `/var/log/cups/error_log` | Ошибки и предупреждения демона |

## lpadmin — настройка принтеров

`lpadmin` добавляет, изменяет и удаляет принтеры.

| Параметр | Описание |
|---|---|
| `-p ИМЯ` | Имя принтера (при добавлении/изменении) |
| `-L "РАСПОЛОЖЕНИЕ"` | Строка с описанием расположения |
| `-v URI` | URI устройства (например `socket://192.168.1.100:9100`) |
| `-m МОДЕЛЬ` | PPD-модель из базы драйверов |
| `-x ИМЯ` | Удалить принтер |
| `-u allow:user` / `-u deny:user` | Управление доступом пользователей |
| `-o параметр=значение` | Установить параметр принтера |

```bash
# Добавить сетевой принтер
lpadmin -p myprinter -L "Офис 3B" -v socket://192.168.1.100:9100 -m drv:///sample.drv/laserjet.ppd

# Удалить принтер
lpadmin -x myprinter
```

### lpoptions — принтер по умолчанию

```bash
lpoptions -d ИМЯ_ПРИНТЕРА    # установить принтер по умолчанию
```

## lpr — печать файлов (интерфейс BSD)

`lpr` отправляет файлы в очередь печати.

```bash
lpr file.pdf
lpr -P laserjet file.pdf
```

| Параметр | Описание |
|---|---|
| `-P ПРИНТЕР` | Указать принтер назначения |
| `-# N` | Напечатать N копий |
| `-o landscape` | Печать в альбомной ориентации |
| `-o two-sided-long-edge` | Двусторонняя печать, переворот по длинному краю (портрет) |
| `-o two-sided-short-edge` | Двусторонняя печать, переворот по короткому краю (ландшафт) |
| `-o media=A4` | Размер бумаги |
| `-o collate=true` | Раскладывать копии по порядку |
| `-o page-ranges=1-5` | Диапазон страниц |
| `-o fit-to-page` | Масштабировать под страницу |
| `-o outputorder=reverse` | Обратный порядок вывода |

---

## lp — печать файлов (интерфейс System V)

Команда `lp` является альтернативой `lpr` в стиле System V.

```bash
lp file.pdf
lp -d laserjet -n 2 file.pdf
```

| Параметр | Описание |
|---|---|
| `-d ПРИНТЕР` | Указать принтер назначения |
| `-n N` | Количество копий |
| `-o параметр=значение` | Параметры принтера (те же, что у `lpr -o`) |

## lpstat — статус принтеров

```bash
lpstat -p           # список принтеров и их состояние
lpstat -d           # принтер по умолчанию
lpstat -v           # URI устройств
lpstat -o           # все задания печати
lpstat -a           # статус приёма заданий
lpstat -t           # вся информация о статусе
```

## Управление заданиями печати

### lpq — просмотр очереди

```bash
lpq             # очередь принтера по умолчанию
lpq -a          # очереди всех принтеров
lpq -P laserjet # очередь конкретного принтера
```

### lprm — удаление заданий

```bash
lprm 42         # удалить задание 42
lprm -          # удалить все свои задания
```

### cancel — отмена заданий (System V)

```bash
cancel 42              # отменить задание 42
cancel laserjet-42     # отменить по формату принтер-задание
```

### lpmove — перемещение заданий

```bash
lpmove laserjet-42 colorjet    # переместить задание 42 с laserjet на colorjet
```

## Управление очередями принтеров

### Отклонение и принятие заданий

`cupsreject` запрещает добавление новых заданий в очередь (уже стоящие задания продолжают печататься):

```bash
cupsreject laserjet
cupsaccept laserjet
```

### Отключение и включение

`cupsdisable` останавливает обработку заданий принтером (очередь продолжает принимать новые задания):

```bash
cupsdisable laserjet
cupsenable laserjet
```

Для полного отключения принтера использовать обе команды:

```bash
cupsreject laserjet    # прекратить принимать новые задания
cupsdisable laserjet   # прекратить обрабатывать существующие задания
```

## Типичные вопросы экзамена

1. Как называется демон CUPS? → `cupsd`
2. Какой файл — основной файл конфигурации CUPS? → `/etc/cups/cupsd.conf`
3. Какой файл конфигурации CUPS нельзя редактировать при работающем CUPS? → `/etc/cups/printers.conf`
4. Какая директива включает веб-интерфейс CUPS? → `WebInterface Yes` в `cupsd.conf`
5. По какому адресу доступен веб-интерфейс CUPS? → `http://localhost:631`
6. В каком файле журнала CUPS записывается каждая напечатанная страница? → `/var/log/cups/page_log`
7. Какая команда добавляет сетевой принтер `laserjet`? → `lpadmin -p laserjet -v socket://IP:9100 -m МОДЕЛЬ`
8. Какая команда устанавливает принтер по умолчанию? → `lpoptions -d ИМЯ_ПРИНТЕРА`
9. Как напечатать 3 копии `file.pdf` на принтере `office`? → `lpr -P office -# 3 file.pdf` или `lp -d office -n 3 file.pdf`
10. Какой параметр `lpr` включает двустороннюю печать с переворотом по длинному краю? → `-o two-sided-long-edge`
11. Как проверить очередь всех принтеров? → `lpq -a`
12. Какая команда удаляет все ожидающие задания текущего пользователя? → `lprm -`
13. Какая команда System V отменяет задание печати? → `cancel JOBID` или `cancel ПРИНТЕР-JOBID`
14. Что делает `lpmove laserjet-5 colorjet`? → Перемещает задание 5 с принтера `laserjet` на `colorjet`.
15. В чём разница между `cupsreject` и `cupsdisable`? → `cupsreject` прекращает добавление новых заданий в очередь. `cupsdisable` останавливает обработку заданий, уже находящихся в очереди.
16. Какая команда показывает состояние всех принтеров? → `lpstat -p`
17. Какая команда показывает принтер по умолчанию? → `lpstat -d`
18. Какой файл обеспечивает совместимость CUPS с LPD? → `/etc/printcap`

## Упражнения

### Практическое упражнение 1 — Добавление принтера

Добавьте принтер с именем `hplaser`, расположенный в «Серверной», с URI устройства `socket://10.0.0.5:9100`.

<details>
<summary>Ответ</summary>

```bash
lpadmin -p hplaser -L "Серверная" -v socket://10.0.0.5:9100
```

На практике также потребуется указать PPD-модель (`-m`) для выбора нужного драйвера.

</details>

---

### Практическое упражнение 2 — Печать с параметрами

Напечатайте страницы 2–4 файла `report.pdf` в альбомной ориентации на принтере `office`.

<details>
<summary>Ответ</summary>

```bash
lpr -P office -o page-ranges=2-4 -o landscape report.pdf
```

</details>
